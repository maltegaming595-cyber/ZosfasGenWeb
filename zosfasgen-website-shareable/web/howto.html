How to use

Everything here is synced with the Discord bot. Use the dashboard for the same functions as bot commands.

1) Login Click Login and accept the Discord prompt.
2) Generator Search by appid or game name. The bot posts an embed with a download button to the files channel.
3) Tickets Choose a topic (Game / Premium / Glitches / Other). Closing locks the ticket. Tickets auto-delete after 24h.
4) Premium / Boost Check your status and remaining time. If you don’t have access, you’ll be redirected to the correct Discord channel.
5) Giveaways Join giveaways in Discord. Admins can create them in the dashboard and they auto-end on time.
If you ever see “access denied” after login, just click Login again — it happens when the permission prompt is cancelled.
